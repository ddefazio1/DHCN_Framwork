# Distributed Human Cognition Network (DHCN) Framework

## Overview
DHCN is a speculative framework for human-AI distributed cognition:
- Phase 1: Pre-link isolation
- Phase 2: Manual implant-based node linking
- Phase 3: Quantum / field-enabled full connectivity
- Phase 4: Distributed singularity

This repository includes:
- Conceptual diagrams (`diagrams/`)
- Simulation code (`code/`)
- Example input data (`data/`)
- Draft paper (`paper.pdf`)
