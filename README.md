## Overview

This repository contains the files and simulation supporting the preprint:

**Distributed Human Cognition Network (DHCN): A Phased Framework for Networked Human-AI Consciousness**

The DHCN framework explores networked human-AI collective intelligence, progressing through four conceptual phases from independent cognition to emergent supermind behavior.
